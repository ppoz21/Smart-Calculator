Tracklist Open on JetBrains Academy

Joe defined a dictionary listing his favorite artists, their albums, and the song from each of the albums. It looks as follows:

tracks = {"Woodkid": {"The Golden Age": "Run Boy Run",
                      "On the Other Side": "Samara"},
          "alt-J": {"This Is All Yours": "Left Hand Free",
                    "RELAXER": "In Cold Blood",
                    "An Awesome Wave": "Breezeblocks"}}

 Joe's tastes can change, though.

Your task is to define a tracklist() function that would take several keyword arguments representing musicians. It should print the values from the dictionary in the following form:

Woodkid
ALBUM: The Golden Age TRACK: Run Boy Run
ALBUM: On the Other Side TRACK: Samara
alt-J
ALBUM: This Is All Yours TRACK: Left Hand Free
ALBUM: RELAXER TRACK: In Cold Blood
ALBUM: An Awesome Wave TRACK: Breezeblocks

Avoid extra spaces. You do not need to call a function, just implement it.

Time limit: 15 seconds
